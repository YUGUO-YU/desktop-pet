# 🐱 桌面精灵 - 加菲猫 使用说明书

## 一、基本信息

- **软件名称**：桌面精灵 V2.0
- **形象**：3D 加菲猫（抱臂傲娇姿势）
- **版本**：2.0.0

## 二、功能介绍

### 1. 基础互动

| 功能 | 操作方式 | 说明 |
|------|----------|------|
| 摸头 | 点击 🤚 按钮 | 加菲猫会开心地说"喵~ 摸头好舒服！" |
| 喂食 | 点击 🍪 按钮 | 喂食后心情值 +1，好感度提升 |
| 玩耍 | 点击 🧶 按钮 | 一起玩耍，心情值 +1 |

### 2. 触摸互动

- **点击加菲猫身体**：触发痒痒反应
- 不同部位有不同回复：
  - 头：摸头好舒服
  - 身体：痒痒~
  - 手臂：这是我的手臂
  - 脚脚：别挠我脚脚

### 3. 心情系统

- 心情值 1-5 级
- 通过喂食、玩耍提升心情
- 心情越高，加菲猫越活跃

## 四、使用方法

### 方法一：直接打开（推荐）

1. 双击 `test.html` 文件
2. 浏览器中打开即可使用

### 方法二：Electron 桌面应用

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版
npm run build
```

## 五、注意事项

1. 确保浏览器支持 SVG 图像
2. 首次加载可能稍慢，请耐心等待
3. 点击按钮后等待几秒看回复

## 六、常见问题

**Q: 为什么点不动？**
A: 确保已完全加载完成

**Q: 心情值是什么？**
A: 代表加菲猫的情绪状态，越高越开心

**Q: 可以移动位置吗？**
A: 拖动顶部灰色条可移动（Electron 版本）
